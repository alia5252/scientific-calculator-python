# Scientific Calculator (Python)

A command-line Scientific Calculator built using Python. This calculator performs basic arithmetic operations as well as scientific calculations such as trigonometric functions, logarithms, factorials, powers, and square roots. It also stores calculation history in a text file.

## Features

### Basic Operations
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Power (**)
- Modulus (%)

### Scientific Operations
- Square Root (sqrt)
- Sine (sin)
- Cosine (cos)
- Tangent (tan)
- Logarithm (log)
- Factorial (fact)

### Extra Features
- Calculation History
- Save History to File
- Clear History Option
- Input Validation
- User-Friendly Menu System

## Technologies Used

- Python
- Math Module
- File Handling
- Exception Handling

## How to Run

1. Clone the repository

```bash
git clone https://github.com/alia5252/scientific-calculator-python.git
```

2. Open the project folder

```bash
cd scientific-calculator-python
```

3. Run the program

```bash
python calculator.py
```
