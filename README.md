# Number Guessing Game 🎮

A fun terminal-based Number Guessing Game built using Python.

## Features
- Easy, Medium, and Hard difficulty levels
- Score system
- High score saving using file handling
- Hint mode
- Cheat mode
- Input validation
- Remaining attempts system
- Replay option

## Technologies Used
- Python
- File Handling
- Random Module
- Loops and Conditions

## How to Run

```bash
python game.py
